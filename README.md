
## Loan-calculator

### Getting Started

1. **Clone repo**

    `git clone https://github.com/codinglawyer/loan-calculator`

2. **Install dependencies**

    `npm install`

3. **Start dev server**

    `npm run dev`

4. **Open your browser**

    `http://localhost:8080/`
